### Installation 安装

> 不支持 Windows 10 2004 以下版本

Windows 10 2004 及以上版本:

+ [在 Microsoft Store 中下载](https://www.microsoft.com/store/productId/9MWX740P5W6N)
+ 或安装 Release 中的安装包
